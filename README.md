# Yoru

Yoru — мобильное приложение интернет-магазина одежды, разработанное в рамках лабораторных работ по дисциплине «Мобильная разработка».

## Описание проекта

Приложение показывает пользователю каталог товаров с категориями и списком позиций. Данные о товарах загружаются из локального JSON-файла.

В рамках проекта реализуются:

- загрузка данных из JSON;
- отображение категорий товаров;
- отображение списка товаров;
- навигация между экранами приложения;
- использование архитектурного подхода с ViewModel.

## Стек и платформа

- Android
- Kotlin
- Jetpack Compose
- Android SDK
- ViewModel
- Gson

## Сборка проекта

Чтобы запустить проект:

1. Склонируйте репозиторий.
2. Откройте проект в Android Studio.
3. Дождитесь синхронизации Gradle.
4. Запустите приложение на эмуляторе или Android-устройстве.

## Структура проекта

- `model` — модели данных каталога;
- `assets` — JSON-файл с товарами;
- `ViewModel` — хранение и обработка данных;
- `UI` — пользовательский интерфейс на Jetpack Compose.
