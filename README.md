# Yoru

**Yoru** — мобильное Android-приложение с каталогом одежды, разработанное командой `team-sepulki`.

Приложение позволяет просматривать товары по категориям, смотреть новинки и переходить между основными разделами через нижнюю навигацию.

## Описание проекта

Yoru представляет собой прототип мобильного магазина одежды.

В приложении реализованы:

- каталог товаров;
- разделение товаров по категориям;
- отдельная вкладка с новинками;
- карточки товаров с изображением, названием и ценой;
- нижняя навигация между каталогом и корзиной;
- загрузка данных о товарах из локального JSON-файла.

Каталог содержит 50 товаров в 5 категориях:

- джинсы;
- футболки;
- рубашки;
- обувь;
- верхняя одежда.

## Технологии

В проекте используются:

- Kotlin;
- Android;
- Jetpack Compose;
- Material 3;
- Gradle Kotlin DSL;
- Gson для чтения JSON-данных;
- Coil для загрузки изображений товаров.

## Требования для сборки

Для запуска проекта понадобятся:

- Android Studio;
- JDK 11 или новее;
- Android SDK;
- Android-эмулятор или физическое Android-устройство.

Параметры проекта:

- `minSdk`: 24;
- `targetSdk`: 36;
- `compileSdk`: 36;
- `applicationId`: `com.yoru.app`.

## Инструкция по сборке

### 1. Клонировать репозиторий

```bash
git clone https://github.com/FEIP-FEFU-Mobile-Spring-2026/team-sepulki.git
```

### 2. Перейти в папку проекта

```bash
cd team-sepulki
```

### 3. Открыть проект в Android Studio

1. Запустите Android Studio.
2. Нажмите **Open**.
3. Выберите папку проекта `team-sepulki`.
4. Дождитесь завершения Gradle Sync.

Если Android Studio предложит установить недостающие компоненты SDK или обновить Gradle, согласитесь с установкой.

### 4. Запустить приложение

1. Выберите устройство для запуска: эмулятор или подключенный Android-телефон.
2. Нажмите **Run** в Android Studio.
3. После сборки приложение установится и откроется на выбранном устройстве.

## Сборка через командную строку

Для сборки debug-версии можно использовать Gradle Wrapper.

На Windows:

```bash
gradlew.bat assembleDebug
```

На macOS / Linux:

```bash
./gradlew assembleDebug
```

После успешной сборки APK-файл будет находиться по пути:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Данные приложения

Данные каталога хранятся в файле:

```text
app/src/main/assets/products.json
```

В JSON-файле описаны категории и товары: название, описание, цена, ссылка на изображение, теги и принадлежность к категории.
