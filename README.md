# Yoru

Мобильное приложение интернет-магазина одежды, разработанное в рамках лабораторных работ по дисциплине «Мобильная разработка».

## Описание проекта

Приложение предоставляет пользователю каталог товаров с возможностью просмотра категорий и списка товаров. Данные о товарах загружаются из локального JSON-файла.

В рамках лабораторной работы реализуются:

* загрузка данных из JSON;
* отображение категорий товаров;
* отображение списка товаров;
* навигация между экранами приложения;
* использование архитектурного подхода с ViewModel.

## Используемые технологии

* Kotlin
* Android SDK
* Jetpack Compose
* ViewModel
* Gson

## Сборка проекта

Для запуска проекта необходимо:

1. Клонировать репозиторий.
2. Открыть проект в Android Studio.
3. Выполнить синхронизацию Gradle.
4. Запустить приложение на эмуляторе или Android-устройстве.

## Структура проекта

* `model` — модели данных каталога;
* `assets` — JSON-файл с товарами;
* `ViewModel` — хранение и обработка данных;
* `UI` — пользовательский интерфейс на Jetpack Compose.
