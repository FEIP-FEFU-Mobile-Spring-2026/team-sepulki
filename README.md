# Yoru

Мобильное приложение интернет-магазина одежды, разработанное в рамках лабораторной работы №2 по дисциплине «Мобильная разработка».

## Описание проекта

В рамках лабораторной работы реализован экран каталога товаров с использованием архитектурного подхода MVVM.

Функциональные возможности:

* загрузка данных из локального JSON-файла;
* отображение категорий товаров;
* автоматическое формирование категории «Новинки» на основе тега `New`;
* отображение списка товаров по выбранной категории;
* отображение карточек товаров;
* нижняя навигация между экраном каталога и корзиной;
* хранение данных в ViewModel.

## Используемые технологии

* Kotlin
* Android SDK
* Jetpack Compose
* ViewModel
* Gson

## Сборка проекта

Для запуска проекта необходимо:

1. Клонировать репозиторий.
2. Открыть проект в Android Studio.
3. Выполнить синхронизацию Gradle.
4. Запустить приложение на эмуляторе или Android-устройстве.

## Структура проекта

* `model` — модели данных каталога;
* `assets` — JSON-файл с товарами;
* `CatalogViewModel` — хранение и обработка данных;
* `ui` — пользовательский интерфейс приложения.
